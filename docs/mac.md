# Mac-specific notes

## Requirements

**Xcode**:

Install the latest XCode from the App Store. Open it at least once
so it installs the command line tools.

**Git/rsync**

Install via Homebrew or similar tool.

## Audio

To play audio, use Homebrew to install mpv and lame.
